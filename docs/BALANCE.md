# Novus — Intentional balance ledger

Deliberate gameplay decisions that *look* like bugs or gaps. **Check here before proposing a "fix."** Backfilled 2026-06-06 from session memory; now authoritative. Design pillars live in SYNOPSIS.md; this file covers system-level calls.

- **Powah cables transfer 1:1 with cell transfer rate** — Create-style "plumbing" philosophy, not Powah tiering. Charging `maxChargeRate` is keyed to this ladder (snowball 10k = hardened). Don't re-tier.
- **`biomancy_organic_drops` farm yield** — no `killed_by_player` gate; farms work. Confirmed intentional.
- **AgriCraft is mid/late-game optional** — ships its own tutorial; vanilla + Farmer's Delight is the early path. Don't tutorialize or gate it earlier.
- **Cradle ("Flesh Crucible") tribute expansions are gameplay-driven** — pitch new registrations only on an observed in-game gap, not for completeness.
- **DT seed→sapling is one-way** (reverse removed 2026-05-30) — deliberate seed-economy sink; don't re-add sapling→seed.
- **DT dirt bucket disabled** (`generateDirtBucketRecipes=false`) — `dt_seed_sapling_recipes.js` is the sole conversion path.
- **DT trees don't drop Supplementaries ash** — custom destruction path bypasses Moonlight; accepted 2026-05-09; don't propose unsolicited.
- **Quark Simple Harvest is hoe-only** — right-click harvest+replant requires a hoe; bare-hand off by intent.
- **Scarecrows' Territory spawner defaults** — left at mod defaults intentionally (scarecrows don't suppress spawners).
- **Chocolate-bar tiering** — `forge:bars/chocolate` = Create + compatdelight milk only; dark/blazing are separate tiers.
- **Seed pressing economy** — Create compacting turns seeds into XP/oil/slimes etc.; exists to give Mega seeds a use, not to be "balanced away."
- **Sleep never skips night — MULTIPLAYER ONLY** (`playersSleepingPercentage 101` via `kubejs/server_scripts/default_gamerules.js.disabled`, 2026-06-07) — ships inert, same pattern as `seasons.toml.disabled`; rename to `.js` on a server + one restart to enable. Protects the 1-season-per-real-week Serene Seasons clock (SS 9.x advances seasons by dayTime delta; each skipped night = ~10 real min of drift). Once applied it persists in level.dat, so update-flow reverting the filename doesn't revert the world. Singleplayer/local worlds deliberately keep vanilla sleep (forgiving, player-manipulable). Accepted side effects when enabled: thunderstorms and Comforts hammock day-skips also blocked; beds still set spawn/reset phantoms.
- **Subtle Effects — underwater chest particles + biome ambient particles off** (1.1.0) — `chestsOpenRandomlyUnderwaterFrequency=0` (no random underwater chest-opening puffs) and `biomeParticlesRadius=0` (no biome ambient particles). Deliberate visual restraint, not a broken/disabled mod; the `biome_color_water_particles` built-in pack stays on. Don't "re-enable for completeness."
- **Clearer Water disabled by default** (1.1.0) — VanillaTweaks `ClearerWater.zip` removed from the default `resourcePacks` order but kept bundled in `resourcepacks/`. Players opt back in via Options → Resource Packs. Synced into `config/defaultoptions/options.txt` so new installs honor it (instance `options.txt` alone is dev-machine-only).
- **Simple Clouds ceiling pinned to Y322** (2026-06-14) — `clientSideCloudHeight = 322` in `config/simpleclouds-client.toml` (default 128). Source-verified mapping: cloud mesh base = world-Y `clientSideCloudHeight`, growing upward at CLOUD_SCALE=8, so the lowest possible cloud block bottoms out exactly at this value. 322 = highest placeable block (Y319) + 2-block air gap. Intentional, not a misconfig — don't revert to 128. Two deliberate consequences: (1) clouds sit at the world ceiling so they look very high/small from ground; (2) clouds become physically unreachable in survival, which sidesteps the SC↔Oculus in-cloud "out-of-bounds" fog rendering glitch. Ships as a packwiz index override (already refreshed), NOT via DefaultOptions — it's a mod TOML, so the per-user regen caveat is moot once the pack places the file.
